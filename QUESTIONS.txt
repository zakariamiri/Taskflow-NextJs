Q1 : Différence React vs Next.js

React (Vite) :
Routing dans le code (App.tsx)
utilise react-router-dom
tout côté CLIENT

Next.js :
Routing = dossiers (app/)
pages automatiques
support SERVER (SSR)

Différence principale : Next.js utilise routing automatique par dossiers

Q2 :
Next.js : 1 seul fichier 
React : 3 choses ( composant , route dans App.tsx , import )

=> Next.js est plus simple

Q3 : 
React → côté CLIENT
Next.js → côté SERVEUR (plus rapide + sécurisé)

Q5 :
React : useState, useEffect, fetch, setState, loading

Next.js : seulement ( const res = await fetch(...) , const data = await res.json() )
beaucoup plus simple
